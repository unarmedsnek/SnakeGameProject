Snake Game Project

Description
This is a classic Snake game implemented in Python using the Pygame library.
The player controls a snake that moves around a playing area, trying to eat food pellets to grow longer.
The game ends if the snake collides with the walls or with its own body.
The game features a main menu, a scoring system, and a game over screen with a high score display.

Requirements

To run this game, you will need:
Python 3.x
Pygame library

Setup and Installation

1.  Ensure Python is installed:** If you don't have Python, download and install it from [python.org](https://www.python.org/).
2.  Clone or download the project:** Make sure you have all the project files, including the `assets` folder.
3.  Navigate to the project directory:** Open a terminal or command prompt and change to the directory where you saved the project.

    cd path/to/SnakeGameProject

4.  Install Pygame:
    It's recommended to use a virtual environment. You can install Pygame using pip:

    pip install pygame

    Alternatively, if a `requirements.txt` file is provided:

    pip install -r requirements.txt

How to Run the Game
Once the setup is complete, you can run the game by executing the `main.py` script from the project's root directory:

python main.py
